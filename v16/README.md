# URC-Ground-Station
This repository contains the ground station software for the University Rover Challenge (URC). It provides the interface and control logic required to communicate with the rover's onboard systems, including the Jetson Orin Nano and various avionics components.

## Installation
This project uses **Conda** to manage dependencies. Ensure you have Miniconda or Anaconda installed before proceeding.

1. **Clone the Repository:**
   ```bash
   git clone [https://github.com/CalculateX/URC-Ground-Station.git](https://github.com/CalculateX/URC-Ground-Station.git)
   cd URC-Ground-Station

2. **Configure the Environment**
Run the `setup_env.bat` file.

3. **Reinstall if Dependencies have changed***
Run the `reinstall.bat` file.

## Use
To open the GUI, run the `launch_site.bat` file.

## Maintenance
To make any changes, edit the `app.py` file.
